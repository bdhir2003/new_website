Projects

FairCare: Evaluating Fairness in Health Risk Prediction Models Using Statistical Machine Learning

This project focuses on predicting 30-day hospital readmission for diabetic patients using the UCI Diabetes dataset (20,000 samples). I implemented machine learning models such as logistic regression, random forest, and SVM to replicate and extend results from recent healthcare studies. A key part of the project was evaluating fairness, following the Mishra & Khazanchi (2025) fairness framework, to identify disparities in predictions across demographic groups like age, gender, and race. The project highlights how predictive models can be both accurate and fair, contributing to more ethical and inclusive healthcare AI systems.
